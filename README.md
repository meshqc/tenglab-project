# TengLab-project
## Liver Cancer Liquid Biopsy - RNA-seq Analysis

### Dataset Information
- **Study**: RNA-seq analysis of liver cancer patients' plasma
- **GEO Accession**: [GSE142987](https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE142987)
- **Samples**: 65 total (35 liver cancer, 30 healthy donors)
- **Technology**: HiSeq X Ten (Homo sapiens)
- **Publication**: [PMID: 33391469](https://pubmed.ncbi.nlm.nih.gov/33391469)

### Data Files
- `GSE142987_sample_count_matrix.txt.gz` - Raw gene expression counts
- `GSE142987_series_matrix.txt.gz` - Sample metadata
